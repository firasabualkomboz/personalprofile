How to add simple search to your Laravel website?

Search form :

First, create a form search

< form action="/search" method="POST" role="search">
{{ csrf_field() }}
< input type="text" name="q" placeholder="@lang('auth.search')">
< button type="submit" class="site-btn">@lang('auth.search_btn')< /button>
< /form>

Search action :

!! Dont forget use - App\Models\Product; and use Illuminate\Support\Facades\Input;

Route::any('/search',function(){
$q = Input::get ( 'q' );
$product = \App\Models\Product::where('name','LIKE','%'.$q.'%')
->orWhere('price','LIKE','%'.$q.'%')->get();
if(count($product) > 0)
return view('front.search_result')
->withDetails($product)->withQuery ( $q );
else return view ('front.search_result')->withMessage('No Details found. Try to search again !');
});
